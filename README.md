# Test Case Generator

A simple web-based tool to generate basic software test cases from feature descriptions.

## Description
This application allows users to enter one or more feature descriptions and generate corresponding test cases.  
It is designed for learning purposes and to demonstrate frontend logic using JavaScript.

## Features
- Enter multiple features (one per line)
- Generates structured test cases for each feature
- Supports common features like Login, Signup, and Search
- Copy generated test cases to clipboard
- Clean and simple user interface

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

## How to Use
1. Enter feature names in the text area (one per line)
2. Click **Generate Test Cases**
3. View generated test cases below
4. Use **Copy Test Cases** to copy them if needed

## Project Type
Frontend practice project for understanding:
- DOM manipulation
- Conditional logic
- User interaction handling
